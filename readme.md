## Install

```bash
$ npm install --global timer
```

## CLI

```
$ timer --help

  Options
    --secs=<number of seconds>

  Examples
    $ timer --name=5

    ╓───────────────────────────────╖
    ║     Timer for 00h:00m:05s     ║
    ║     Remaining 00h:00m:05s     ║
    ╙───────────────────────────────╜
```
